To use this website just click notes onto the piano roll and select how many notes you want to generate with the slider.
Then click the button with the big G on it. This will generate and add the new notes onto the piano roll. (This process will work better if a single sequence of notes is used rather than chords).

You can clear notes using the red button.
You can play the music if you hit the play button!!

The website generates the notes using a "Long Short Term Memory" neural network which is a type of Recurrent Neural Network.
It was created using TensorFlow which is an open source machine learning API.

Enjoy!!!
